# CSE519: Data Science Fundamentals Quiz App

Static HTML/CSS/JavaScript quiz application for CSE519.

## Features

- Topic-based quizzes
- Immediate answer feedback
- Final score report
- Incorrect answer review
- Mobile responsive
- GitHub Pages compatible

## Deployment

Works with:
- GitHub Pages
- Netlify
- Vercel
- VSCode Live Server

Do NOT open with file:/// locally because browser module imports throw a tantrum.
